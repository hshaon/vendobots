# VendoBots

VendoBots integrates a mobile vending robot with projection-based cues, multimodal interaction, and autonomous navigation.  
It combines a customer mobile app, robot interface, ROS 2 control, backend coordination, and HRI evaluation framework.

## Modules
- mobile_app/: Food ordering app (Android/iOS)
- robot_interface/: Robot-mounted Android UI with projections and speech
- ros_ws/: ROS 2 packages (navigation, perception, voice, projection)
- backend_server/: API, database, MQTT communication
- evaluation/: Data collection and metric analysis
- docs/: Reports and design files
- scripts/: Setup utilities
